What is this setup?

OpenClaw is an AI gateway running on your Mac Mini (MMM4). It connects Claw ‚Äî your AI agent ‚Äî to your messaging channels, tools, and memory. You talk to Claw from any device; OpenClaw routes everything behind the scenes.

The 4-tier model system means Claw automatically picks the right model for the task: fast and capable for most things, expensive+smart only when it matters, cheap for trivial work.

Full System Architecture ‚Äî How Claw Works
How a message flows
  1. You send a message on Telegram / iMessage / Discord / Webchat
  2. OpenClaw gateway receives it and routes to main session
  3. Claw (Sonnet 4.6) reads context + memory and decides what to do
  4. If complex ‚Üí spawns a sub-agent with the right model tier
  5. Sub-agent runs, returns structured output
  6. Claw synthesizes and replies to you
  7. Durable decisions written to memory.md
